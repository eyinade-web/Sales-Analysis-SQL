## Sales Data Analysis Using SQL
This project analyzes a sales dataset using SQL.
I used SQL queries to answer business questions and generate insights from the data.

## Business Questions Answered
Which salesperson generated the highest revenue?
What is total sales by country?
Which product sold the most?
What is the average sale amount per salesperson?
How many boxes were shipped last month?

## Key Insight
Australia generated the highest total sales.
One product consistently had the highest total boxes shipped.
Sales dropped between February and May before rising again in June.
